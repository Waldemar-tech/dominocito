🔒 7 · Seguridad — Capas Activas en Cada Ruta
🚦 Rate Limiting
  • Login: 5 req/min/IP
  • Registro: 3 req/min/IP
  • Apuestas: 10 req/min
  • 429 Too Many Requests
🪪 JWT Auth
  • Todas las rutas protegidas
  • Header: Authorization Bearer
  • Payload: user_id + role
  • Expiry configurable
🔐 Integridad DB
  • SELECT FOR UPDATE en wallets
  • Anti race condition en apuestas
  • Transacciones atómicas
  • bcrypt para passwords
🛡️ HTTP Security
  • Helmet.js headers
  • Input sanitization
  • CORS configurado
  • XSS / injection protection
📊 Sorteo Auditable
  • Seed guardado antes del sorteo
  • Fórmula determinista pública
  • dc_banca_log para auditoría
  • Cada acción tiene timestamp
